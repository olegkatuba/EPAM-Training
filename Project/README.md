### How to use

1. To run project use ```npm run start```
1. Build via ```npm run build```
1. Build for production via ```npm run release```

### Used separators

| Separator     | Is used to                    |
| ------------- | ----------------------------- |
| `-`           | separate words                |
| `__`          | separate blocks and elements  |
| `--`          | separate modificators         |
